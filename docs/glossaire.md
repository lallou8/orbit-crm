# GLOSSAIRE MÉTIER - ORBIT CRM

| Terme | Définition |
|-------|------------|
| **Client** | Entreprise qui utilise les services ORBIT Smart Monitoring |
| **Admin ORBIT** | Employé interne qui gère les clients, le SAV et les projets |
| **Ticket SAV** | Réclamation ou demande d'assistance technique émise par un client |
| **Intervention** | Déplacement d'un technicien chez le client pour résoudre un problème |
| **Fiche d'intervention** | Compte-rendu détaillé d'une intervention (technicien, date, actions) |
| **Licence** | Droit d'utilisation d'un logiciel ORBIT avec date d'expiration |
| **Facture** | Document de facturation émis ou prévu pour un client |
| **Projet** | Installation ou déploiement d'une solution ORBIT chez un client |
| **Réclamation** | Synonyme de ticket SAV (vue client) |
| **Tableau de bord** | Vue synthétique avec indicateurs et statistiques |
